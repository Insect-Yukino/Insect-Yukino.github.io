如何处理线程池中发生的异常？

如果我们使用 `execute()` 提交任务，发生异常后会直接从工作线程中抛出来。

如果线程设置了 `UncaughtExceptionHandler`，就能捕获到；如果没有，通常会打印到控制台。

`submit()` 异常不会直接往外抛，而是会被封装进 `Future` 里。只有当我们调用 `get()` 时，才会拿到 `ExecutionException`。

1. 通常我们应该在任务内部自己捕获异常，对异常做显示的处理。 
2. 如果我们使用 `submit()` 时，一定要处理 `Future`

> `execute()` 异常会直接抛给工作线程
>
> `submit()` 异常会封装到 `Future` 里，需要 `get()` 才能拿到

也可以给线程工厂设置 `UncaughtExceptionHandler`

重写线程池 `afterExecute()` 钩子方法

```java
import java.util.concurrent.*;
import java.util.concurrent.atomic.AtomicInteger;

public class ThreadPoolConfig {

    public static ExecutorService newExecutor() {
        ThreadFactory threadFactory = new ThreadFactory() {
            private final AtomicInteger count = new AtomicInteger(1);

            @Override
            public Thread newThread(Runnable r) {
                Thread t = new Thread(r, "biz-thread-" + count.getAndIncrement());
                t.setUncaughtExceptionHandler((thread, ex) -> {
                    System.err.println("未捕获异常, thread=" + thread.getName());
                    ex.printStackTrace();
                });
                return t;
            }
        };

        return new ThreadPoolExecutor(
                4,
                8,
                60,
                TimeUnit.SECONDS,
                new LinkedBlockingQueue<>(1000),
                threadFactory,
                new ThreadPoolExecutor.AbortPolicy()
        ) {
            @Override
            protected void afterExecute(Runnable r, Throwable t) {
                super.afterExecute(r, t);

                Throwable ex = t;
                if (ex == null && r instanceof Future<?>) {
                    try {
                        Future<?> future = (Future<?>) r;
                        if (future.isDone()) {
                            future.get();
                        }
                    } catch (CancellationException ce) {
                        ex = ce;
                    } catch (ExecutionException ee) {
                        ex = ee.getCause();
                    } catch (InterruptedException ie) {
                        Thread.currentThread().interrupt();
                    }
                }

                if (ex != null) {
                    System.err.println("线程池任务异常");
                    ex.printStackTrace();
                }
            }
        };
    }

    public static void main(String[] args) {
        ExecutorService executor = newExecutor();

        executor.execute(() -> {
            throw new RuntimeException("execute 异常");
        });

        executor.submit(() -> {
            throw new RuntimeException("submit 异常");
        });

        executor.shutdown();
    }
}
```
