`DelayQueue` 是 Java 并发包里的一个无界队列，里面只能放实现了 `Delayed` 接口的元素。并且只有任务到期之后，才能从队列中取出来，所以它特别适合做：

- 延迟任务
- 超时取消
- 订单超时关闭
- 缓存过期处理
- 若干秒执行某事

它底层是一个按到期时间排序的优先队列，对头永远是最早到期的那个元素，但即使队头有元素，只要还没有到期，也拿不出来。调用 `DelayQueue#take` 时，如果还没到期，线程就会阻塞等待。

`DelayQueue` 中的元素必须实现 `Delayed` 接口，因为 `DelayQueue` 必须知道两件事情：

1.  这个元素什么时候到期
2. 多个元素之间谁先到期
