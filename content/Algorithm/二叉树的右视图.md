这是[力扣](https://leetcode.cn/problems/binary-tree-right-side-view/)的官网解法。思路如下：

1. 题目中的意思是每层只取最右侧的节点值
2. 维护一个 `HashMap` 用来存放每层的深度和其对应的值。维护一个 `maxDept`，用来最后遍历结果
3. 维护两个栈，其中一个是用来层序遍历节点。另一个栈用来存对应节点的深度
4. 遍历整个树，取出每层最右边的节点放入 `HashMap` 中
5. 最终将 `HashMap` 的值放入结果集

```java
class Solution {
    public List<Integer> rightSideView(TreeNode root) {
        // 每一个深度对应一个值
        Map<Integer, Integer> rightmostValueAtDepth = new HashMap<Integer, Integer>();
        // 记录最大深度
        int maxDepth = -1;
        // 遍历节点
        Deque<TreeNode> nodeStack = new LinkedList<TreeNode>();
        // 深度栈
        Deque<Integer> depthStack = new LinkedList<Integer>();
        // 先将根节点放入遍历栈
        nodeStack.push(root);
        // 开始深度 0
        depthStack.push(0);
        // 如果栈中有节点
        while (!nodeStack.isEmpty()) {
            // 取出节点
            TreeNode node = nodeStack.pop();
            // 取出当前深度
            int depth = depthStack.pop();
            // 如果节点不为空
            if (node != null) {
                // 记录最大深度
                maxDepth = Math.max(maxDepth, depth);
                // 如果该深度没有值
                if (!rightmostValueAtDepth.containsKey(depth)) {
                    // 记录该值
                    rightmostValueAtDepth.put(depth, node.val);
                }
                // 存入对应层的节点，要先存左再存右，否则的话会先取出左侧
                nodeStack.push(node.left);
                nodeStack.push(node.right);
                // 存入对应层的深度
                depthStack.push(depth + 1);
                depthStack.push(depth + 1);
            }
        }
        List<Integer> rightView = new ArrayList<Integer>();
        // 将遍历的结果存入结果集
        for (int depth = 0; depth <= maxDepth; depth++) {
            rightView.add(rightmostValueAtDepth.get(depth));
        }
        return rightView;
    }
}
```

> 注意遍历结果集的时候，因为 `maxDept` 的大小正好是整个集合的数量。所以边界要理清。
>
> 如果要求给出左视图，那 `HashMap` 集合中应该保存每层最左侧的节点。存入逻辑就应该是
>
> ```java
> nodeStack.push(node.right); // 先放最右的节点
> nodeStack.push(node.left); // 再放最左的节点
> ```

该方法的深度优先搜索，它是先遍历最右侧的节点。直到节点为空。再遍历左侧。每次只长朝一个方向遍历。
