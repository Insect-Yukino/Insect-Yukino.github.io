在启动 Nacos 后，我看到控制台打印出的前端地址不是平常的 `localhost`，而是：

```bash
 Console: http://192.168.182.1:8848/nacos/index.html
```

这是因为 Nacos 在启动时会自动选一个本机可对外通信的 IP 作为 Console 地址。

它的规则大致是：

1. 扫描所有网卡
2. 排除 `127.0.0.1`
3. 选一个 **非回环、非虚拟的 IPv4**
4. 打印出来作为 Console 访问提示

因为我在内网，所以这个地址就是本机的内网地址。
