对象结构和很多其他知识有关，比如锁机制，哈希，对象大小、数组等。

Java 对象有三部分：

- 对象头
- 对象体
- 对其字节

## 对象头

对象头包括三个字段，第一个字段叫做 Mark Word（标记字段）

第二个字段叫做 Class Pointer 对象指针

压缩指针技术

第三个字段叫做 Array Length

## 对象体

对象体包含对象的实例变量
